Projet : Analyseur de Données Météo

Ce projet permet d’analyser des données météorologiques stockées
dans un fichier CSV. Les données sont lues à l’aide d’un itérateur
personnalisé afin d’optimiser l’utilisation de la mémoire.

Fonctionnalités :
- Lecture des données météo depuis un fichier CSV
- Détection des anomalies de température et de pluie
- Calcul de la température moyenne
- Utilisation de la programmation orientée objet
- Utilisation des fonctions map, filter et reduce

Exécution :
python main.py
